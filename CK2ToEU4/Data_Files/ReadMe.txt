This program will convert a Crusader Kings 2 save game to a Europa Universalis 4 save game.

----
IMPORTANT LINKS
----

Forum thread: http://forum.paradoxplaza.com/forum/ --- 
Github project: https://github.com/ParadoxGameConverters/CK2ToEU4
Windows development build: https://github.com/ParadoxGameConverters/CK2ToEU4/releases/tag/windows_development_build

----
REQUIREMENTS
----
CK2 - Conclave DLC active
EU4 - Dharma DLC active

-----
INSTRUCTIONS
-----
1. Run ConverterFrontend.exe.
2. Fill out all fields and keep clicking Next.


----
MANUAL METHOD INSTRUCTIONS
----

1. Download the converter and unzip it wherever.
2. Copy your CK2 save into the converter directory.
3. Edit configuration.txt to point at your game installs, your savegame location, and to set any options you care about.
4. Run CK2ToEU4Converter.exe
5. Go into the output folder, then the subfolder named after your save.
6. Copy the folder named after your save and the corresponding mod file into your EU4 mod folder.
7. Start EU4, and select the mod.
